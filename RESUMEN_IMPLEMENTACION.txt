## 🔌 API ENDPOINTS (20+ Endpoints)

### Products
GET    /api/products              ✅ Lista todos
GET    /api/products/:id          ✅ Obtiene uno
POST   /api/products              ✅ Crea uno
PUT    /api/products/:id          ✅ Actualiza
DELETE /api/products/:id          ✅ Elimina

### Categories
GET    /api/categories            ✅ Lista
GET    /api/categories/:id        ✅ Obtiene
POST   /api/categories            ✅ Crea
PUT    /api/categories/:id        ✅ Actualiza
DELETE /api/categories/:id        ✅ Elimina

### Users
GET    /api/users                 ✅ Lista
GET    /api/users/:id             ✅ Obtiene
POST   /api/users                 ✅ Crea
PUT    /api/users/:id             ✅ Actualiza
DELETE /api/users/:id             ✅ Elimina

### Orders
GET    /api/orders                ✅ Lista
GET    /api/orders/:id            ✅ Obtiene
POST   /api/orders                ✅ Crea
PUT    /api/orders/:id            ✅ Actualiza
DELETE /api/orders/:id            ✅ Elimina

### Cart
POST   /api/cart                  ✅ Crea carrito
GET    /api/cart/:cartId          ✅ Obtiene carrito
POST   /api/cart/:cartId/items    ✅ Agrega item
PUT    /api/cart/:cartId/items/:id✅ Actualiza qty
DELETE /api/cart/:cartId/items/:id✅ Elimina item
DELETE /api/cart/:cartId          ✅ Vacía carrito

## ⚡ SOCKET.IO EVENTOS EN TIEMPO REAL

### Client → Server (Emits)
✅ join_cart            - Unirse a sala de carrito
✅ add_to_cart         - Agregar producto
✅ update_item         - Cambiar cantidad
✅ remove_from_cart    - Quitar producto
✅ clear_cart          - Vaciar carrito
✅ get_cart            - Obtener datos actuales

### Server → Client (Broadcasts)
✅ item_added          - Cuando se agrega un producto
✅ item_updated        - Cuando cambia la cantidad
✅ item_removed        - Cuando se quita un producto
✅ cart_cleared        - Cuando se vacía el carrito
✅ cart_data           - Enviando datos del carrito
